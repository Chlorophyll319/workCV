## 技術改進

1. **單一數據源原則**：所有 UI 都以 `fileSystem.ts` 為唯一配置來源
2. **自動同步機制**：修改配置文件時，界面自動反映變更
3. **程式碼簡潔性**：移除冗餘邏輯，提高可維護性
4. **跨瀏覽器相容**：捲動軸隱藏適用於所有主流瀏覽器
5. **響應式設計**：保持完整的 RWD 功能
