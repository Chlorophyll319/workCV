### 3. 技術問題修復

#### Icon 系統問題
**問題**: @iconify/vue 組件語法錯誤，使用了錯誤的 `name` 屬性
**解決方案**: 
- 批量修復所有 `name=` 為 `icon=`
- 統一使用 `heroicons:` 格式圖示
- 建立專案 icon 映射函數處理 `i-ph-*` 格式轉換

#### CSS 重複警告
**問題**: `text-gray-800` 和 `text-base` 重複定義
**解決方案**: 移除重複的 `text-base` class

#### HTML 結構錯誤
**問題**: HeroSectionNew.vue 有多餘的 `</div>` 標籤
**解決方案**: 修復 HTML 標籤匹配問題
