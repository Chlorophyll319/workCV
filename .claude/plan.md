# 履歷網站重設計計畫（Modern Infographic Resume · Blue–Orange）

> 目標：把現有的 VSCode 暗色主題＋垂直佈局，重構為 **明亮底 + 藍色主色 + 橘色重點** 的「現代商務資訊圖表風」，與你的 PDF 履歷視覺一致，兼顧可讀性＆RWD。

---

## 1. 為什麼要改

- **品牌一致**：網站風格與 PDF 履歷一致（大標粗體、色塊分區、藍橘識別）。
- **HR 可掃讀**：大量文字改為分塊與清單，3–30 秒快速抓重點。
- **差異化**：從工程感（VSCode 暗色）改為 **專業＋活力** 的商務簡報感。

---

## 2. 視覺規範（Design System）

基於 `.claude\參考風格.png` 的實際設計風格：

### 色彩（精確對應參考圖）

- **Base/背景**：`#FFFAE6`（米白奶茶色調）
- **主色（Blue）**：`#2196F3`（用於大標題 YEH, PENG、區塊標題）
- **強調色（Orange）**：`#FF9800`（用於職稱標籤、圖示、重點標記）
- **文字**：`#000000`（主文字）、`#424242`（輔助文字）
- **分隔線**：`#000000`（細實線，用於區塊分隔）

### 字體層級

- **大標題（姓名）**：超大粗體，藍色 `#2196F3`
- **職稱標籤**：橘色背景 `#FF9800` + 黑字，圓角矩形
- **區塊標題**：藍色中標題 `#2196F3`，粗體
- **內文**：黑色常規字重
- **技術標籤**：小字，可用細框線或淡背景

### 版型結構（對應參考圖）

- **左欄（30%）**：
  - 姓名大標 + 職稱橘標籤
  - 簡介段落（3-4行）
  - 技能/專長（四大分類，條列式）

- **右欄（70%）**：
  - 聯絡方式（橘色圖示 + 文字）
  - 專案經驗（每項包含標題、描述、技術點）
  - 學歷（簡潔條列）

### 視覺元素

- **圖示**：橘色填充，簡約風格（電話、信箱等）
- **標籤**：橘色背景圓角矩形（如 Front-end）
- **分隔**：黑色細實線，清楚劃分區塊
- **間距**：充足的白空間，避免擁擠感

## 3. 版型（Layout）與 RWD

### 桌機（≥1024px）

- **雙欄版型**（對應參考圖比例）：
  - 左欄（約 30% 寬）：**姓名大標＋職稱標籤、簡介、技能分類**
  - 右欄（約 70% 寬）：**聯絡方式、專案經驗、學歷**
- **滾動時**：左欄可 sticky（到 footer 前停止）
- **分隔線**：左右欄間用黑色細實線分隔

### 平板（768–1023px）

- 轉為 **上中下**：Header → 左欄卡片 → 右欄內容（各區塊順序保持語意）

### 手機（<768px）

- **單欄直排**：所有區塊全寬；標題條、徽章與間距放大，避免擁擠

### 間距

- 區塊：`py-12`（手機 `py-8`）
- 卡片內：`p-6`（手機 `p-4`）
- 網格：`gap-6`（手機 `gap-4`）

   - 右欄維持你既有的專案卡資料結構；左欄新增 Profile/Contact/Skills 三卡。
4. **Typography 調整**
