# 履歷網站重設計計畫（Modern Infographic Resume · Blue–Orange）

> 目標：把現有的 VSCode 暗色主題＋垂直佈局，重構為 **明亮底 + 藍色主色 + 橘色重點** 的「現代商務資訊圖表風」，與你的 PDF 履歷視覺一致，兼顧可讀性＆RWD。

---

## 1. 為什麼要改

- **品牌一致**：網站風格與 PDF 履歷一致（大標粗體、色塊分區、藍橘識別）。
- **HR 可掃讀**：大量文字改為分塊與清單，3–30 秒快速抓重點。
- **差異化**：從工程感（VSCode 暗色）改為 **專業＋活力** 的商務簡報感。

基於 `.claude\參考風格.png` 的實際設計風格：

- **大標題（姓名）**：超大粗體，藍色 `#2196F3`

## 3. 版型（Layout）與 RWD

### 桌機（≥1024px）

- **雙欄版型**（對應參考圖比例）：
  - 左欄（約 30% 寬）：**姓名大標＋職稱標籤、簡介、技能分類**
  - 右欄（約 70% 寬）：**聯絡方式、專案經驗、學歷**
- **滾動時**：左欄可 sticky（到 footer 前停止）
- **分隔線**：左右欄間用黑色細實線分隔

### 平板（768–1023px）

- 轉為 **上中下**：Header → 左欄卡片 → 右欄內容（各區塊順序保持語意）

### 手機（<768px）

- **單欄直排**：所有區塊全寬；標題條、徽章與間距放大，避免擁擠

### 間距

- 區塊：`py-12`（手機 `py-8`）
- 卡片內：`p-6`（手機 `p-4`）
- 網格：`gap-6`（手機 `gap-4`）

## 12. 【最新修復】重複 class 屬性錯誤修復（2025/09/09）

### 問題發現
在進行 CSS 變數到 Tailwind 語義化類名轉換時，發現了重複 class 屬性的語法錯誤：

```html
<!-- 錯誤示例 -->
<div 
  class="bg-white rounded-xl border-2 p-6..."
  class="border-primary"  <!-- 重複的 class 屬性 -->
>
```

### 修復過程
系統性地修復了所有 Vue 組件中的重複 class 屬性錯誤：

#### 修復檔案清單
1. **AboutSection.vue**：修復 VSCode 檔案標籤的 border 顏色
2. **SkillsSection.vue**：修復 4 個技能分類卡片的樣式
   - Frontend 和 Tools：`border-primary`、`bg-primary`、`text-primary`
   - Backend 和 AI：`border-accent`、`bg-accent`、`text-accent`
3. **ProjectsSection.vue**：修復專案星號圖示和 GitHub 按鈕顏色
4. **HeroSectionNew.vue**：修復 GitHub 和聯絡按鈕樣式

### 轉換成果
✅ 成功將所有 CSS 變數轉換為語義化 Tailwind 類名：
- `style="color: var(--color-primary);"` → `class="text-primary"`
- `style="border-color: var(--color-primary);"` → `class="border-primary"`
- `style="background-color: var(--color-accent);"` → `class="bg-accent"`

### 開發伺服器狀態
✅ 修復後開發伺服器成功重啟，運行在 `http://localhost:3005/`
✅ 所有 Vue 語法錯誤已修復，no TypeScript/template 錯誤

## 14. 【最新優化】手機版互動邏輯修正（2025/09/09）

### 手機版側邊欄修正
修正了手機版的互動邏輯問題，將側邊活動列的按鈕功能分離：

#### 問題背景
- 原本手機版的漢堡選單和 folder 按鈕都控制同一個側邊欄
- 這導致互動邏輯混亂，使用者體驗不佳

#### 解決方案
**功能分離：**
- **漢堡選單按鈕**：控制頂部選單項目的下拉顯示（Design、Code、AI-Powered、Projects、Marketing、Connect）
- **folder 按鈕**：控制側邊欄檔案總管面板的顯示

#### 手機版頂部選單實作
- **新增狀態**：`isMobileMenuExpanded` 控制手機版選單下拉
- **選單樣式**：類似右鍵選單的浮動效果
  - 絕對定位：`absolute top-[30px] left-2`
  - 白色背景：`bg-white rounded-md shadow-lg`
  - 高層級：`z-50` 覆蓋在主要內容上方
- **互動優化**：點擊選單項目後自動關閉下拉選單

#### 響應式標題優化
同時修正了手機版標題的 RWD 問題：
- **小螢幕**（<640px）：顯示簡短版本「Evelyn Resume」
- **大螢幕**（≥640px）：顯示完整版本「葉芃 (Evelyn) - Frontend Engineer Resume」
- **防止換行**：`whitespace-nowrap` + `text-ellipsis` 確保單行顯示

#### 附加功能
**refresh 按鈕實作：**
- 點擊側邊欄的 refresh 圖示可重新整理頁面
- 使用 `window.location.reload()` 實現真正的頁面重載

**資料夾收合功能：**
- RESUME-PROJECT 資料夾可點擊收合/展開
